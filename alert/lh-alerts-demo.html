LH Alerts — LearnHub Notification System

Library notifikasi & dialog resmi LearnHub LMS. Toast, dialog, confirm, prompt, loading, dan inline alert dengan animasi smooth yang sesuai design system admin panel.

import lh-alerts.js  →  LH.toast()  ·  LH.dialog()  ·  LH.confirm()
1
Toast Notifications
LH.toast() · LH.success() · LH.error() · LH.warning() · LH.info()
Success
Aksi berhasil — data tersimpan, upload selesai, dll.
LH.success('Data tersimpan!');
Error
Aksi gagal — koneksi error, validasi gagal, dll.
LH.error('Koneksi gagal!');
Warning
Peringatan — sesi hampir habis, data tidak lengkap.
LH.warning('Sesi hampir habis!');
Info
Informasi — pembaruan sistem, notifikasi umum.
LH.info('Pembaruan tersedia');
Teal
Aksi khusus — proses berjalan, notifikasi sistem.
LH.toast({type:'teal',message:...});
Purple
Notifikasi premium — badge, pencapaian, sertifikat.
LH.toast({type:'purple',duration:6000});
Indigo
Notifikasi modul — komentar, kursus baru, artikel.
LH.toast({type:'indigo'});
Orange
Notifikasi media — upload, storage, media library.
LH.toast({type:'orange'});

2
Dialog Modal
LH.dialog() — Promise<value>
Info Dialog
Dialog informasi umum dengan satu tombol tutup.
LH.dialog({ type: 'info', icon: 'shield', title: 'Informasi Akses', buttons: [{label:'Mengerti', close:true}] });
Success Dialog
Konfirmasi setelah aksi berhasil diselesaikan.
LH.dialog({ type: 'success', icon: 'save', title: 'Tersimpan!', buttons: [{label:'Lihat',...}, ...] });
Warning Dialog
Dialog peringatan untuk aksi yang berisiko atau terbatas.
LH.dialog({ type: 'warning', icon: 'lock', title: 'Akses Terbatas', });
Multi-Action Dialog
Dialog dengan beberapa tombol — Promise resolve ke value masing-masing.
const action = await LH.dialog({ buttons: [ {label:'Draft', value:'draft'}, {label:'Publish', value:'publish'} ] });
Klik untuk lihat hasil Promise...
3
Confirm Dialog
LH.confirm() — Promise<boolean>
Hapus Data
Konfirmasi sebelum menghapus — resolve true/false.
const ok = await LH.confirm({ type: 'error', icon: 'delete', confirmLabel: 'Ya, Hapus', confirmClass: 'lh-btn-danger' });
Menunggu konfirmasi...
Konfirmasi Logout
Konfirmasi sebelum pengguna keluar dari sesi.
LH.confirm({ type: 'warning', icon: 'logout', title: 'Keluar dari LearnHub?', confirmClass: 'lh-btn-warning' });
Menunggu konfirmasi...
Konfirmasi + Checkbox
Tombol konfirmasi aktif hanya setelah checkbox dicentang.
LH.confirm({ requireCheck: true, checkLabel: 'Saya memahami...' });
Menunggu konfirmasi...
Reset Password
Konfirmasi kirim link reset password ke email.
LH.confirm({ type: 'info', icon: 'key', confirmLabel: 'Kirim Link Reset', confirmClass: 'lh-btn-primary' });
Menunggu konfirmasi...
4
Prompt Input Dialog
LH.prompt() — Promise<string|null>
Input Sederhana
Input teks — resolve ke string yang dimasukkan user.
const val = await LH.prompt({ title: 'Nama Kategori Baru', label: 'Nama', placeholder: 'cth: Tutorial Laravel' });
Menunggu input...
Input + Validasi
Input dengan custom validate function — pesan error inline.
LH.prompt({ validate(v) { if(!v.includes('@')) return 'Email tidak valid' } });
Menunggu input...
Rename / Edit Cepat
Prompt dengan nilai default — untuk fitur rename kursus/artikel.
LH.prompt({ title: 'Rename Kursus', defaultValue: 'Laravel 11 Fullstack', confirmLabel: 'Simpan Nama' });
Menunggu input...
Input Password
Input password dengan validasi kekuatan — min 8 karakter + huruf besar + angka.
LH.prompt({ inputType: 'password', validate: v => v.length < 8 ? 'Min 8 karakter' : null });
Menunggu input...
5
Loading Overlay
LH.loading() — { close(), setMessage(), setTitle() }
Loading Dasar
Loading overlay full-screen — auto close setelah 2.5 detik.
const ld = LH.loading({ title: 'Memuat Data...' }); // Lalu: ld.close();
Loading Dinamis
Update pesan loading saat proses berlangsung via setMessage().
ld.setTitle('Mengunggah...'); ld.setMessage('File 2/3 selesai');
Loading + Batalkan
Loading yang bisa dibatalkan user — closable: true.
LH.loading({ title: 'Memproses...', closable: true });
6
Inline Alert
LH.alert(selector, opts) — inject ke dalam elemen DOM
Inject ke Elemen — Cocok untuk form validation
Alert yang diinjeksikan ke dalam elemen — ideal untuk feedback dalam form, halaman kursus, atau panel admin.
7
Skenario Nyata LMS
Alur lengkap dari aksi admin sehari-hari
Simpan Kursus
Loading → API call → Success dialog → Toast kembali.
// loading → sukses → toast const ld = LH.loading(...); // await API... ld.close(() => LH.dialog(...));
Hapus Pengguna
Confirm → Loading → Success toast atau Error toast.
if(await LH.confirm(...)){ const ld = LH.loading(...); ld.close(() => LH.success(...)); }
Approve Instruktur
Prompt catatan → Confirm → Loading → Success dialog.
const catatan = await LH.prompt(...); if(catatan !== null && await LH.confirm(...)){ // proses approval... }
Publish Kursus
Multi-dialog (Draft/Publish) → Loading → Success atau Info toast.
const action = await LH.dialog({ buttons: [ {value:'draft',...}, {value:'publish',...} ] });
